Features

Everything you need,
nothing you don't.

Built for people who compare Amazon products against Temu. No accounts, no subscriptions — open a product page and click.

⚡
One-click scrape
Hit Scrape Page and ASIN, price and brand populate instantly — no manual searching through the page required.
📋
Smart copy buttons
Two separate copy buttons: ASIN + Price together (fills 2 cells), Brand alone (fills 1 cell). Paste exactly where you need without overwriting other columns.
🌍
16 Amazon domains
Works on amazon.com, .in, .co.uk, .de, .ca, .com.au, .co.jp, .fr, .es, .it, .nl, .se, .pl, .ae, .sa and .sg — one extension for every marketplace.
💰
Any currency
Handles USD, INR, GBP, EUR and every other Amazon currency — extracts just the number so your spreadsheet stays clean and consistent.
🌙
Dark mode built-in
Toggle between light and dark with one click. Your preference is saved so it persists across every session.
🛡️
Zero data collected
Everything runs locally in your browser. No servers, no tracking, no analytics. Your product research stays yours.
How it works

Three steps.
Done.

1
Install the extension
Download the zip, unzip it, go to your browser extensions page and load unpacked. Takes under 60 seconds.
chrome://extensions → Load unpacked
2
Open any Amazon product page
Navigate to a product on any Amazon domain and let the page fully load. Then click the extension icon in your toolbar.
3
Scrape → Copy → Paste
Hit Scrape Page, then use the two copy buttons. Click the ASIN cell in your spreadsheet → Ctrl+V for ASIN+Price. Click the Brand cell → Ctrl+V for Brand.
Two targeted pastes. Zero overwrites.

Install

Get it on your browser.

Available for Chrome and Firefox. Both use the same extension package.

🌐
Google Chrome
Load unpacked via chrome://extensions or install from the Chrome Web Store once published.
→
🦊
Firefox
Available on Firefox Add-ons (addons.mozilla.org). Same zip, fully Manifest V3 compatible.
→
Manual install (developer mode)
1. Download the .zip file from GitHub
2. Unzip into a folder
3. Go to chrome://extensions
4. Enable Developer Mode (top right)
5. Click Load unpacked → select the folder
6. Open any Amazon product page and click the extension icon 📦
